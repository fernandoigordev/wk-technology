### Como rodar a aplicação :pager:
  
  1. Todos os arquivos necessários estão na pasta "exe" na raiz do repositório
  2. Restaure o banco de dados no MySql utilizando o dump com o nome de "dump-wktechnology.sql"
  3. Configure a conexão com o banco no arquivo Configuracao.ini
  4. Na pasta exe está disponível a biblioteca do banco de dados junto com a aplicação!
